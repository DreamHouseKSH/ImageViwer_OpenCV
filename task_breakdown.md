
4. 진행률 자동 계산:
   - 완료된 작업은 `[x]`로 표시

## 📊 진행률 계산 규칙

1. 각 하위 작업은 동일한 가중치를 가짐
2. 모든 하위 작업이 완료되어야 상위 작업이 완료된 것으로 간주
3. 진행률 = (완료된 작업 수 / 총 작업 수) * 100
